##  一、環境建置與啟動方法

### 1. 配置金鑰
* 在 `backend` 資料夾下建立 **`.env`** 檔案。
* 寫入 Google Gemini 的 API Key (可至 Google AI Studio 免費申請)：
  ```text
  GEMINI_API_KEY=你的金鑰字串



### 2. 安裝依賴套件

* 開啟終端機 (建議使用 Anaconda Prompt)。
* 建立 Conda 虛擬環境
```bash
conda create --name hw01_env python=3.10

```
* 啟動虛擬環境 (請根據自身資料夾名稱修改)：
```bash
conda activate hw01_env

```


* 進入 `backend` 目錄。
* 執行以下指令安裝所需套件：
```bash
pip install fastapi uvicorn python-dotenv google-genai

```



### 3. 啟動伺服器

* 執行以下指令啟動 FastAPI：
```bash
uvicorn main:app --reload

```


*  **注意：** 測試期間 **請勿關閉** 此終端機視窗。

### 4. 開啟網頁

* 直接雙擊開啟 **`frontend/index.html`** 檔案，即可開始使用。

##  二、五大核心功能測試指南

### 1. LLM 模型切換測試 與 Streaming 串流輸出測試

* **操作：** 於左側下拉選單切換模型。
* **輸入提示詞：** > 「請給我一篇 1000 字的繁體中文文章，介紹人工智慧的歷史。」
* **驗證：** 觀察畫面是否呈現 **逐字輸出的打字機效果**，且模型能正常切換並回覆(Gemini 2.5 flash-lite輸出速度會明顯快於Gemini 2.5 flash)。

### 2. System Prompt 測試

* **操作：** 於左側 System Prompt 欄位輸入以下人設：
> 「你現在是一位脾氣暴躁的資深程式設計師，對於使用者的問題極度不耐煩，且每句話都必須以『菜鳥，聽好了！』作為開頭。」


* **驗證：** 發送任意問題，驗證模型回覆是否 **嚴格遵守上述語氣設定**。

### 3. Temperature 測試

* **操作：** 測試前請先 **重新整理網頁** 清空歷史紀錄。接著輸入以下提示詞：
> 「請幫我發明 3 個科幻小說裡的外星人名字，並簡述特徵。」


* **驗證：** 將 Temperature 調整至不同數值測試，觀察 **低溫時的穩定性** 與 **高溫時的隨機創意發散**。

### 4. 交談短期記憶測試

* **操作 (第一回合)：** 傳送以下背景資訊：
> 「我最喜歡的水果是香蕉，而且我的寵物是一隻叫做『雷神』的烏龜。」


* **操作 (第二回合)：** 待模型回覆後，緊接著詢問：
> 「根據我們剛剛的對話，請問我養了什麼寵物？牠叫什麼名字？那我喜歡吃什麼？」


* **驗證：** 系統能 **精準回覆上一回合的記憶內容**。

# HW02

實作了以下四個主要功能，以及一些畫面上的優化

###   1.Long-term memory

* **操作：** 傳送訊息後重新開啟或重新整理網頁
* **驗證：** 對話紀錄還會留存，並且支援清除對話紀錄

###   2.Multimodal

* **操作：** 點擊輸入框旁的圖片上傳按鈕，上傳一張圖片並輸入：「請詳細描述這張圖片裡面有什麼？」
* **驗證：** 前端會顯示圖片預覽縮圖，後端會將圖片轉為 Base64 傳送給 Gemini，模型能精準辨識並回覆圖片內容細節。
  
###   3.Auto routing between models

* **操作：** 將左側模型選擇設定為 Auto。
> 情境 A： 輸入純文字「請給我一個簡短的笑話。」
> 情境 B： 上傳圖片並詢問內容，或詢問天氣資訊。

* **驗證：** 觀察對話框上方的 系統日誌 (System Log)：
> 情境 A 會顯示：⚙️ [Auto Routing] 一般對話，分配至極速模型 Gemini 2.5 Flash。
> 情境 B 會顯示：⚙️ [Auto Routing] 偵測到「多模態圖片/工具呼叫」，自動切換至強邏輯模型 Gemini 2.5 Pro。

###   4.Tool use, MCP

* **操作：** 測試系統調用外部 API 的能力，請分別輸入以下問題：
> 「請問現在系統的精確時間是幾點幾分？」
> 「請問現在台北市的天氣如何？溫度大概幾度？」
> 「請幫我查詢現在BTC(支援虛擬貨幣)的最新價格是多少？」

* **驗證：** 系統會中斷一般文字生成，觸發 Function Call。回傳由 Open-Meteo 或 Binance API 獲取的真實即時數據。

###   小優化

* 1.當對話送出後，發送鍵會轉為停止鍵，支援中止模型當前的操作

* 2.當選定上傳的照片時並且尚未送出對話，會有縮小版的圖片喜善在對話框之上
